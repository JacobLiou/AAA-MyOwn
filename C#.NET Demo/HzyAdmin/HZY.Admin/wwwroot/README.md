# hzy-admin-ui

#### 介绍
用浏览器引入elementui组件库方式，写一套后台模板页面！

注意：请使用服务端方式打开页面

例如打开地址：http://127.0.0.1:5500/views/main.html 或者 http://localhost:5500/views/main.html 这样得形式
